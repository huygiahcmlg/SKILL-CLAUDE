# SEO Frameworks - Tài Liệu Tham Khảo

## Phân Loại Search Intent (TINS)

| Intent | Định nghĩa | Dấu hiệu trong keyword | Loại nội dung phù hợp |
|--------|-----------|----------------------|----------------------|
| **Transactional (Giao dịch)** | Muốn mua/đăng ký ngay | mua, giá, order, đặt, thuê | Landing page, trang sản phẩm |
| **Informational (Thông tin)** | Muốn học/tìm hiểu | là gì, cách, hướng dẫn, tại sao | Blog, hướng dẫn, FAQ |
| **Navigational (Điều hướng)** | Tìm thương hiệu/trang cụ thể | tên brand, đăng nhập, website | Homepage, trang sản phẩm |
| **Commercial (Thương mại)** | Đang cân nhắc mua | tốt nhất, so sánh, review, top | Listicle, so sánh, review |

## Công Thức Ước Tính Word Count Từ SERP

Dựa trên loại nội dung và độ cạnh tranh:

| Loại nội dung | Word count khuyến nghị |
|---------------|----------------------|
| Bài hướng dẫn chi tiết | 2,000 - 4,000 từ |
| Listicle (danh sách) | 1,500 - 2,500 từ |
| So sánh sản phẩm | 1,500 - 3,000 từ |
| Tin tức / cập nhật | 500 - 1,000 từ |
| Landing page | 800 - 1,500 từ |
| FAQ page | 1,000 - 2,000 từ |

## Framework Phân Tích SERP

### Bước 1: Đọc Kết Quả Organic Top 10
Xác định:
- Loại domain đang thống trị (brand lớn / blog cá nhân / forum)
- Loại nội dung phổ biến nhất (hướng dẫn / listicle / so sánh)
- Có Featured Snippet không? Nội dung gì?

### Bước 2: Phân Tích Title Tags
Pattern title hay gặp cho từng intent:
- **Informational:** "[Keyword] là gì? Hướng dẫn đầy đủ [năm]"
- **Commercial:** "Top 10 [keyword] tốt nhất [năm] | Review chi tiết"
- **Transactional:** "Mua [keyword] giá tốt | Freeship | [Brand]"

### Bước 3: Đọc Snippets
- Snippets tiết lộ nội dung Google ưu tiên hiển thị
- Nếu nhiều snippet dạng list → dùng bullet/number list
- Nếu nhiều snippet dạng đoạn văn → dùng đoạn văn ngắn, súc tích

### Bước 4: Khai Thác People Also Ask
- Mỗi câu hỏi PAA = 1 H2 hoặc H3 tiềm năng
- PAA tiết lộ sub-intent của keyword
- Trả lời đủ 5-7 câu PAA = tăng cơ hội lọt Featured Snippet

## Cấu Trúc Bài Viết Theo Loại Content

### Bài Hướng Dẫn (How-to Guide)
```
H1: Cách [Keyword] - Hướng Dẫn Chi Tiết [Năm]
  Intro (150-200 từ): Trả lời trực tiếp "cách" làm
  H2: [Keyword] là gì? (nếu cần giải thích)
  H2: Tại Sao Cần [Keyword]?
  H2: Cách [Keyword] - Hướng Dẫn Từng Bước
    H3: Bước 1: ...
    H3: Bước 2: ...
  H2: Lưu Ý Quan Trọng Khi [Keyword]
  H2: Câu Hỏi Thường Gặp (FAQ)
  H2: Kết Luận
```

### Bài Listicle (Top/Best)
```
H1: Top [N] [Keyword] Tốt Nhất [Năm] | Đánh Giá Chi Tiết
  Intro (100-150 từ): Tóm tắt lý do chọn list này
  H2: Tiêu Chí Đánh Giá [Keyword]
  H2: Top [N] [Keyword] Tốt Nhất
    H3: 1. [Tên] - [Đặc điểm nổi bật]
    H3: 2. [Tên] - [Đặc điểm nổi bật]
    ...
  H2: Bảng So Sánh Tổng Hợp
  H2: Cách Chọn [Keyword] Phù Hợp
  H2: Câu Hỏi Thường Gặp
  H2: Kết Luận
```

### Bài So Sánh (A vs B)
```
H1: [A] vs [B]: So Sánh Chi Tiết [Năm] - Cái Nào Tốt Hơn?
  Intro (100-150 từ): Tóm tắt ngắn kết quả so sánh
  H2: Tổng Quan Về [A]
  H2: Tổng Quan Về [B]
  H2: So Sánh [A] và [B]
    H3: Tính Năng
    H3: Giá Cả
    H3: Dễ Dùng
    H3: Hỗ Trợ Khách Hàng
  H2: [A] Phù Hợp Cho Ai?
  H2: [B] Phù Hợp Cho Ai?
  H2: Kết Luận: Nên Chọn Cái Nào?
```

## Công Thức Meta Title Hiệu Quả (≤60 ký tự)

| Pattern | Ví dụ |
|---------|-------|
| `[Keyword] - [Lợi ích] \| [Năm]` | Học SEO - Từ Cơ Bản Đến Nâng Cao 2025 |
| `Top [N] [Keyword] [Tính từ] [Năm]` | Top 10 Plugin WordPress Tốt Nhất 2025 |
| `[Keyword] là gì? Hướng Dẫn [Tính từ]` | Backlink là gì? Hướng Dẫn Xây Dựng Hiệu Quả |
| `Cách [Keyword]: [N] Bước [Tính từ]` | Cách Viết Blog: 7 Bước Đơn Giản Cho Người Mới |

## Công Thức Meta Description (≤160 ký tự)
- Mở đầu bằng động từ hành động
- Bao gồm keyword chính
- Có con số cụ thể nếu được
- Kết thúc bằng CTA (Tìm hiểu ngay / Xem chi tiết / Đọc thêm)

Ví dụ: "Khám phá 10 chiến lược SEO hiệu quả nhất 2025. Tăng traffic organic nhanh chóng với hướng dẫn từng bước. Bắt đầu ngay hôm nay!"

## Quy Tắc Keyword Density

- **Keyword chính:** Xuất hiện tự nhiên, không nhồi nhét. 1-2% là ổn.
- **Keyword phụ:** Phân bổ đều trong bài, đặc biệt ở H2/H3
- **Tránh:** Lặp từ khóa quá nhiều trong cùng một đoạn
- **Nên:** Dùng synonym và biến thể tự nhiên của keyword

## Schema Markup Phổ Biến

```json
// FAQ Schema
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "Câu hỏi 1?",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "Câu trả lời 1."
    }
  }]
}

// HowTo Schema
{
  "@context": "https://schema.org",
  "@type": "HowTo",
  "name": "Cách làm gì đó",
  "step": [{
    "@type": "HowToStep",
    "name": "Bước 1",
    "text": "Mô tả bước 1"
  }]
}
```
